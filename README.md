# eclipseplot
A graphical visualization for the ROBUST-RCT tool.

## How to use
Install the dependencies and the package.

    install.packages(c("ggplot2", "dplyr", "ggforce", "cowplot", "tidyr", "magrittr", "readxl"))
    beta <- "https://github.com/pevid-sci/eclipseplot/raw/eclipseplot-(beta)-v0.9.0/eclipseplot_0.9.3.tar.gz"
    install.packages(beta, repos = NULL, type = "source")
