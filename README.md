# Package 'eclipseplot'
A graphical visualization for the *ROBUST-RCT* tool.


## Getting started

Install the beta version
```R
install.packages("eclipseplot_0.9.3.tar.gz", repos = NULL, type = "source")
```

Quick example
```R
library(eclipseplot)
data(sample_brief)
plot <- eclipseplot(sample_brief)
print(plot)
save_eclipseplot(plot)
```

Documentation
```R
?eclipseplot
```
